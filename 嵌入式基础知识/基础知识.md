# TCP/IP
* 应用层：用户接触交互使用的部分，例如http，tfp（文件传输），dns（域名解析）等
* 传输层：传输应用层数据的，例如tcp，udp
* 网络层：处理网络中的数据包，例如IP协议，arp协议
* 数据链路层：连接硬件的部分，如控制网卡，硬件相关的设备驱动
* 物理层：负责传输数据的硬件部分，例如光纤，无线

# 内存分区：
* 堆：存放new或者malloc对象
* 栈：存放局部变量

* 全局存储区：存放全局变量和静态变量
* 常量存储区：存放二进制代码片段
```C语言
char *a; // 栈
a = (char*)malloc(10);  // 分配的10字节空间在堆李
static int b; // 全局存储区
extern char c;     // 全局存储区
strcpy(c, "123");  // 常量区
```
# FREERTOS
* 任务优先级：数字越大，优先级越高，内核花销的内存空间越多
* 栈大小，是bit，不是Bytes
* RT-thread
